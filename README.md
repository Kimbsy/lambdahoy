# lambdahoy

A simple 2D game written in Clojure using Quil.

Entry for the itch.io Spring Lisp Game Jam 2020.

## Usage

```Bash
# Build
lein uberjar

# Run
java -jar target/lambdahoy-0.1.0-SNAPSHOT-standalone.jar
```
